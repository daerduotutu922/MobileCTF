> 另外，严格意义上的CTF其实与实战还是有一些区别，CTF一般其实偏理论，类比护网一般偏现实世界的实战。但是因为移动端领域实在是太小了，所以在这里不再做更细的区分，统称为MobileCTF，只要是移动端的技术都包含在里面。
